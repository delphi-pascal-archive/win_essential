DESCRIPTION
-----------
Component can resize and move any component.


NEW PROPERTIES:
---------------

* Name
  Name of component

* Control
  TWinControl you want modify

* Movable
  Set component movable

* Resizeable
  Set component resizable

* CursorHeight/CursorWidth
  Size of cursor to resiez component

* Color
  Color of cursor

* CursorBevelInner
  Bevel of cursor

* CursorBevelOuter
  Bevel of cursor

* MinHeight/MinWidth
  Size minimal of control

* OnResize
  Event call when control resize
